# Homebrew Tap

A Homebrew tap for installing [wt](https://github.com/agarcher/worktree) - a cross-platform CLI tool for simplified git worktree management.

## Installation

```bash
brew install agarcher/tap/wt
```

Or add the tap first:

```bash
brew tap agarcher/tap
brew install wt
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| wt | Git worktree manager with lifecycle hooks and shell completion |
